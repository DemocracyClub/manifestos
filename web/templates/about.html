About

This project has been rushed together to help me search the manifestos during the TV debates in real time.

I don’t know how interesting it will be in the long term, but in theory I like the idea of a permanent record of what was said.

Adding more documents

I want to add every manifesto I can fine here. I have written a parser for parsing HTML documents in to a hierarchical model.

This means I should be able to add any HTML document to the search/browse pages easily.

If you would like to convert your own documents, please feel free to check out the project on github.

The format is just HTML. I expect every main heading to be an h1 tag. Every heading tag will become a ‘node’ in the document tree, with every non-heading element becoming a child (and lief) of that heading. The parser knows what ‘level’ a heading should be, so if an h2 follows an h3, then the h2 will become an uncle (a sibling of the parent) of the h3.
